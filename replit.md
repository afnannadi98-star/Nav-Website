# NAV - BIM Company Website

## Overview
A professional corporate website for NAV, a Building Information Modeling (BIM) company. The site features a modern, architectural design with a focus on precision and engineering.

## Features
- **Home Page**: Hero section with architectural background, services overview, and featured projects.
- **Services**: Detailed breakdown of BIM coordination, 3D modeling, and scan-to-BIM services.
- **Portfolio**: Dynamic project gallery managed via database.
- **Contact**: Inquiry form that saves submissions to the database.
- **Responsive Design**: Fully mobile-responsive layout.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion, Shadcn UI
- **Backend**: Express.js, Drizzle ORM, PostgreSQL
- **Database**: PostgreSQL (managed via Neon/Replit)

## Project Structure
- `client/`: React frontend code
- `server/`: Express backend code
- `shared/`: Shared types and schemas (Zod/Drizzle)

## Deployment
This project is ready for deployment on Replit.
